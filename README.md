# 曦仔汉字书写助手

一个极简 web app，用来把一句中文拆成单字，展示拼音，并播放单字或整句的笔顺动画。

## 本地运行

```bash
npm install
npm run dev
```

然后打开终端提示的本地地址，默认一般是 `http://127.0.0.1:5173`。

## 当前功能

- 输入一句中文并自动拆成单字
- 点击拼音朗读单字
- 点击“下一笔”逐笔展示当前字
- 点击“播放当前字”连续播放当前字的书写动画
- 左右滑动切换上一个或下一个字
- 点击“连续播放整句话”逐字播放整句
- 支持浏览器语音识别时可直接语音输入

## 项目结构

- `src/main.js`：页面结构与交互逻辑
- `src/styles.css`：页面样式
- `doc/spec.md`：功能规格说明
- `index.html`：Vite 入口页面

## 依赖说明

项目使用 Vite 作为开发服务器与打包工具，页面通过 CDN 引入 `cnchar` 及其 `poly`、`order`、`draw`、`voice` 插件。
